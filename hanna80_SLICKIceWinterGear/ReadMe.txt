Title: SLICK Ice Winter Gear Browser
Description:
	An ASP.NET Core MVC application designed for snowboarders and skiers.
	It allows users to add winter gear to a personal list so they can manage their items.

	To run application:
	* Open project in Visual Studio and press F5, or
	* Open project in Visual Studio Code, press Ctrl+`, and type "dotnet run"

Type: ASP.NET Core MVC
